\# Workspace Boundary



Every record belongs to a workspace.



```text

Workspace

&#x20;в”Ӯ

&#x20;в”ңв”Җв”Җ Connections

&#x20;в”ңв”Җв”Җ API Keys

&#x20;в”ңв”Җв”Җ Tool Calls

&#x20;в”ңв”Җв”Җ Agents

&#x20;в””в”Җв”Җ Audit Logs

```



No data may cross workspace boundaries.
